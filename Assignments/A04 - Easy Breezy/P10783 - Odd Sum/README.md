## 10783 - Odd Sum
### Yoseph Helal 
### Description:

Given a range [a, b], you are to find the summation of all the odd integers in this range. For example,
the summation of all the odd integers in the range [3, 9] is 3 + 5 + 7 + 9 = 24.
